Turnierinfo:

- 16 Spieler / Double-Elimination
- 1 Spiel a 60 Minuten
- ein Spieler hostet das Spiel
- Dazu gelten folgende Einstellungen:
-> Game Type / Spieltyp: Random / Zufall
-> Map Type / Kartentyp: Random / Zufall
-> Map Size / Kartengröße: Medium / Mittel
-> Difficulty / Schwierigkeit: Moderate / Mittel
-> Ressources / Rohstoffe: Default / Standard
-> Population / Bevölkerung: 200
-> Game Speed / Spielgeschwindigkeit: Normal
-> Starting Age / Startzeitalter: Default Age / Standard
-> Reveal Map / Kartenansicht: Normal
-> Victory: Time / Zeitlimit Time: 700 Jahre (1 Stunde)
-> Team Together / Team vereint: Disabled / Haken freilassen
-> Look Speed / Geschw. fest: Enabled / Haken setzen
-> Look Teams / Teams fest: Enabled / Haken setzen
-> Allow Cheats / Cheats zulassen: Disabled / kein Haken
-> All Techs / Alle Techno.: Disabled / kein Haken Record
-> Game / Spiel aufz. Enabled / Haken setzen
-> Die Teutonen werden wegen Ihres klaren Town-Center-Vorteiles nicht zugelassen.
- der Verlierer traegt das Ergebnis ein
- erfolgt die Eintragung nicht rechtzeitig, wird der Gewinner automatisch gelost
Erlaubt sind auch andere Einstellungen, auf die sich beide Spieler einigen.

Cheating:

- Strengstens verboten ist Benutzung von Bugs, z.B. dem Farm-Bug oder dem Fischerboat-Bug. Misachtung führt zur Disqualifizierung für das betreffende Turnier. Grundlage für diese Entscheidung ist dabei die Spielaufzeichnung. jegliche Modifikation ist verboten (auch Skins, Models etc.)
- Disconnects zaehlen als Niederlage, re-joinen ist wenn moeglich erlaubt

Verstoss gegen die Regeln fuehrt zum Ausschluss.
